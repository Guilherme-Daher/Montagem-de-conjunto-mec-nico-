# README - Documentação de Montagem do Kit

## Instruções de Montagem

### 1. Montagem das engrenagens
Encaixe as engrenagens conforme a referência. Utilize a **engrenagem com bico** no meio como referência principal para permitir a rotação das peças.

### 2. Estrutura base
Com as engrenagens alinhadas, feche o compartimento com a tampa para garantir que fiquem fixas e não se soltem durante os movimentos.

### 3. Montagem das gavetas
Encaixe as gavetas na ordem: **220 → 100 → 10 → 10k → 1k**. 
> **Atenção:** Verifique a posição das engrenagens da ponta para garantir o encaixe correto e que a angulação permita o movimento de abertura/fechamento sem obstruções.

### 4. Finalização
Após o encaixe de todos os componentes, posicione a tampa superior para concluir a montagem.
